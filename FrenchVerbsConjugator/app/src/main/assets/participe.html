Participe présent

Présent

Le participe présent exprime une action dans un contexte passé, présent ou futur.

Exemple 1 : J'imaginais des jardiniers arrosant leur plante.

Exemple 2 : J'imagine des jardiniers arrosant leur plante.

Exemple 3 : J'imaginerai des jardiniers arrosant leur plante.

ATTENTION !

Il ne faut pas confondre le participe présent avec l'adjectif verbal. Exemple : Des braises fumantes sur mon tapis. (ici, "fumantes" n'est pas le verbe fumer au participe présent, mais l'adjectif verbal du verbe fumer).

Le participe présent est toujours invariable. Il faut prendre le verbe à la 1ère personne du pluriel (nous) de l'indicatif présent et remplacer la terminaison -ons par -ant.

Exemple avec le verbe gober : nous gobons devient gobant au participe présent.

Exception : le verbe avoir devient sachant au participe présent (mais pas savant).

Verbes du 1er groupe

Au participe présent, les verbes du 1er groupe (comme planter, arroser, ramasser, sécher, rouler, fumer...) ont la conjugaison suivante :

Exemple avec le verbe planter :

plantant

Exemple avec le verbe arroser :

arrosant

Exemple avec le verbe sécher :

séchant

Verbes du 2ème groupe

Au participe présent, les verbes du 2ème groupe (comme vomir, compatir...) ont la conjugaison suivante :

Exemple avec le verbe vomir :

vomissant

Exemple avec le verbe compatir :

compatissant

Verbes du 2ème groupe

Au participe présent, les verbes du 3ème groupe (comme cueillir, tondre, détendre, boire...) ont la conjugaison suivante :

Exemple avec le verbe boire :

buvant

Exemple avec le verbe cueillir :

cueillant

Exemple avec le verbe tondre :

tondant

passé

Le participe passé s'emploie dans de nombreux temps :

passé composé, plus-que-parfait de l'indicatif, passé antérieur, futur antérieur, passé du subjonctif, plus-que-parfait du subjonctif, conditionnel passé, impératif passé.

Exemple 1 : Ils sont arrosés.

Exemple 2 : Elles sont arrosées.

Exemple 3 : Elles ont arrosé.

Exemple 4 : Ils étaient arrosés.

Exemple 5 : Elles étaient arrosées.

Exemple 6 : Elles avaient arrosé.

Exemple 7 : Ils furent arrosés.

Exemple 8 : Elles furent arrosées.

Exemple 9 : Elles eurent arrosé.

Exemple 10 : Ils seront arrosés.

Exemple 11 : Elles seront arrosées.

Exemple 12 : Elles auront arrosé.

Exemple 13 : Qu'ils soient arrosés.

Exemple 14 : Qu'elles soient arrosées.

Exemple 15 : Qu'elles aient arrosé.

Exemple 16 : Qu'ils fussent arrosés.

Exemple 17 : Qu'elles fussent arrosées.

Exemple 18 : Qu'elles eussent arrosé.

Exemple 19 : Ils seraient arrosés.

Exemple 20 : Elles seraient arrosées.

Exemple 21 : Elles auraient arrosé.

Exemple 22 : Ils fussent arrosés.

Exemple 23 : Elles fussent arrosées.

Exemple 24 : Elles eussent arrosé.

Exemple 25 : Soyez arrosé!

Exemple 26 : Ayez arrosé!

Voici les règles de conjugaison des verbes au participe passé.

Participé passé sans auxiliaire :

Le participe passé sans auxiliaire s'accorde en genre et en nombre avec le nom auquel il se rapporte.

Exemple : Les recettes reçues.

Exemple : La fumée répandue.

Le participe passé s'accorde également en fonction d'attribut du sujet.

Exemple : Elle semble fatiguée.

Le participe passé et l'auxiliaire être :

Avec le verbe être, le participe passé s'accorde avec le sujet.

Exemple : Elles sont fatiguées.

Liste des verbes qui utilisent uniquement l'auxiliaire être : naître, mourir, descendre (dans certains sens), redescendre (dans certains sens), monter (dans certains sens), remonter (dans certain sens), venir, advenir, revenir, survenir, convenir, aller, arriver, sortir (dans certains sens), ressortir (dans certain sens), partir, repartir, entrer (dans certains sens), rentrer (dans certains sens), tomber (dans certains sens), rester, tourner et les verbes pronomiaux. Liste des verbes qui utilisent les 2 auxiliaires être ou avoir : apparaître, disparaître, paraître.

Le participe passé est l'auxiliaire avoir :

Le participe passé est invariable avec avoir.

Exemple : Nos graines ont séché.

Exemple : J'ai reçu des graines.

sauf si le verbe est accompagné d'un COD (Complément d'Objet Direct) qui le précède, dans ce cas le participe passé s'accorde en genre et en nombre avec le COD :

Exemple : Les graines que j'ai reçues.

Le participe passé et les verbes essentiellement pronominaux :

Un verbe essentiellement pronominal est un verbe qui n'existe qu'à la forme pronominal comme par exemple le verbe enfuir
(qui est en fait le verbe s'enfuir), se souvenir, s'abstenir...
Ces verbes au participe passé s'accordent en genre et en nombre avec le sujet.

Exemple : Elles se sont enfuies.

Exception : avec les verbes s'arroger et s'entre-nuire qui s'accorde avec le COD si celui-ci est placé avant le verbe.

Le participe passé et les verbes pronominaux ou accidentellement pronominaux :

Le participe passé de ces verbes s'accordent avec le COD si celui-ci est placé avant lui.

Exemple : Il s'est envoyé les graines.

Exemple : Les graines, il se les est envoyées.

Verbes du 1er groupe

Au participe passé, les verbes du 1er groupe (comme planter, arroser, ramasser, sécher, rouler, fumer...) ont la conjugaison suivante, exemple avec le verbe planter :

planté

plantée

plantés

plantées

ayant planté

Verbes du 2ème groupe

Au participe passé, les verbes du 2ème groupe (comme vomir, compatir...) ont la conjugaison suivante, exemple avec le verbe vomir :

vomi

vomie

vomis

vomies

ayant vomi

Verbes du 3ème groupe

Au participe passé, les verbes du 3ème groupe (comme cueillir, tondre, détendre, boire...) ont la conjugaison suivante, exemple avec le verbe boire :

bu

bue

bus

bues

ayant bu
